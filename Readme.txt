WHAT IS HOUSEFLY?
    Housefly Engine is an educational and business project.
    It is written in Python and is currently a CLI-based engine made by one person.
    This project was created with the idea of making a simple, flexible, and readable engine
    that even people with no programming experience can use to create games.
    The project started on 10.03.2026

HOW TO USE IT?
    1. Install Python and IDE like VS
    2. Download folder - Housefly
    3. Open run.py and run *Make sure the path for code is ...\Housefly

FEATURES
    1. Debugger (in works, but it is necessary to navigate trough engine)
    2. Character Creator
    3. Map Creator
    4. Character Editor

Debugger
    To enter debugger, enter - "c housefly"

    Commands:
        "back" - Backing to previous menu
        "exit" - Shuting down program
        "c mouse" - Exiting Debugger

IN WORKS
    1. Event Creator
    2. Event Editor
    3. Map Editor
    4. Run Test game
    5. Create/Save Project
    6. Export Game

-- Q&A --
WHY AM I CREATING GAME ENGINE?
    For educational purposes and for making games. Ofcourse, there are other game engines, but I am not
    satisfied with using them and I find them too confusing. I want to create an engine from 
    scratch and see my progress in both the engine and the games as they develop. Working on this 
    project feels like playing Factorio :D

WHERE DOES THE NAME "HOUSEFLY" COME FROM?
    It originally came from "Fly". I thought there are many engines that are much bigger 
    than my project, and that mine is nothing compared to them. However, I changed it to 
    "Housefly" because there is already a game engine called "Fly".
